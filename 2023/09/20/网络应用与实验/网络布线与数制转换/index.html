网络布线与数制转换

一、信号与传输介质

  1. 信号的定义：是实现信息在网络中传输的一种媒介。
  2. 信号相关概念：
    • 信息：是人们对现实世界事物存在方式或运动状态的一种。例如：数字、文字、图片、声音、视频及动画等。
    • 数据：描述事物相关属性的具体内容。
  3. 信号实现的目的：实现数据的传输与信息共享。
  4. 信号分类：
    • 有线信号和无线信号
    • 数字信号和模拟信号
  5. 信号在传输过程中产生失真的因素：
    • 传输距离——衰减
    • 噪声
    • 速度
    • 其他因素——电磁场等
  6. 数字信号和模拟信号应用场景
    • 数字信号——网络环境比较复杂及干扰比较强的场景
      • 传输距离远
      • 抗干扰能力强
      • 网络传输质量高
    • 模拟信号——网络环境较好及干扰比较弱的场景
      • 传输距离短
      • 抗干扰能力差
      • 网络传输质量低

二、传输介质的连接与布线

  1. 双绞线

    • 从屏蔽类型分类：
      • 屏蔽双绞线
      • 非屏蔽双绞线
    • 从传输速率分类
      • 5类双绞线——100/10MB 网络
      • 超5类双绞线——1000/100MB 网络
      • 6类双绞线——1000/100MB 网络
      • 7类双绞线——10000/1000MB 网络

  2. 光纤

    • 光纤特点：

      • 传输宽带高（10GB/s）
      • 传输距离远（100KM）
      • 抗干扰能力强

    • 光纤分类：

      • 单模光纤
      • 多模光纤

    • 单模光纤与多模光纤区别：

      单模光纤 多模光纤
      用于高速度、长距离 用于低速度、短距离
      成本高 成本低
      端接较难 端接较易
      窄芯线，需要激光源 宽芯线聚光好，光源可采用激光或发光二极管
      耗散小，高效 耗散大，低效

三、以太网接口

  • RJ-45 接口（又称为水晶头）
  • 光纤接口：
    • FC
    • ST
    • SC
    • LC
    • MT-RJ

四、双绞线的连接规范

  • 国际通用的双绞线的标准规范 T568A/T568B（默认使用T568B线序）
  • T568B 线序：橙白、橙、白绿、蓝、白蓝、绿、白棕、棕
  • T568A 线序：1和3，2和6对调即可

五、双绞线（又称网线）的类型

  • 标准网线——两端均是 T568B 或两端均是 T568A（又称为直通线）
  • 交叉网线—— 一端用 T568A，另一端用 T568B
  • 全反线—— 一端是 1-8，另一端 8-1（T568A 或 T568B）（又称为配置线）

六、网络设备的连线规则（常见的网络设备：PC/交换机/路由器）

  • 相同的设备，用交叉线
  • 不同的设备，用直通线
  • PC和路由器相连，用交叉线

七、制作双绞线的步骤

  • 截取适当的长度的网线（0.5M）
  • 用压线钳的剪线口将双绞线的外皮剥去适当的长度
  • 按照线序进行排列，理顺，捋直
  • 用压线钳将线头剪齐，预留长度建议为 1.5厘米
  • 将剪齐的细线插入 RJ-45 接头（水晶头）
  • 用压线钳将其压紧
  • 用测线仪测试双绞线的连通性

八、传输介质

  • 有线传输介质（重点掌握）
    • 双绞线
    • 光纤
  • 无线传输介质（了解）
    • 定义：利用电磁波充当传输媒体的传输介质，包括无线电波、微波、红外线、激光等。

九、综合布线系统（PDS）

  • 定义：用于传输语音、视频、数据、影像和其他信息标准化结构的综合布线系统。
  • 六大子系统
    1. 工作区子系统——连接公司内部网络到信息插座
    2. 水平（配线）子系统——从信息插座到每层楼的管理间子系统的配线架
    3. 管理间子系统——实现每层楼网络互通（网络设备、配线间等）
    4. 垂直子系统——实现所有层网络互通，负责每层管理间和设备间的网络链接
    5. 设备间子系统——主机房（包括：核心设备路由器、交换机、监控系统、电话系统、视频系统、网络系统、有限电视系统等公共系统）
    6. 建筑群子系统——负责楼与楼之间网络信息传输
  • 综合布线的优点：
    • 结构清晰、便于维护和管理
    • 材料统一先进，便于今后的发展需求
    • 灵活性强，适用不同的发展技术
    • 成本低、可靠性高，便于扩展

十、数制转换

  • 计算机数制的相关概念：
    • 数制
    • 数位
    • 基数
    • 位权
  • 常见的进制：
    • 十进制
      • 由十个数字组成 0~9，基数为 10
    • 二进制
      • 由两个数字组成 0 和 1，基数为 2
    • 八进制
      • 由八个数字组成 0~7，基础为 8
    • 十六进制
      • 由十个数字 09 和六个字母 AF 组成 （A 表示 10……F 表示 15），基础 16
  • 数制的表示方法：
    • 十进制数表示方法
      • (1010)₁₀
      • 1010
      • 1010D
    • 二进制数表示方法
      • (1010)₂
      • 1010B
    • 八进制数表示方法
      • (1010)₈
      • 1010O
    • 十六进制数表示方法
      • (1010)₁₆
      • 1010H
  • 数制的计算：
    • 二进制、八进制、十六进制转换成十进制
      • 使用方法：求和法
    • 十进制、八进制、十六进制转换成二级制
      • 使用方法：顺除倒取余
  • 二进制优点：
    • 只有两种数字表示，容易实现
    • 运算规则比较简单
    • 二进制容易实现逻辑运算（真、假；对、错；yes、no）

十一、计算机存储常使用的容量单位

  • 理论计算机容量换算：
    • 1汉字 = 2 bytes（字节）
    • 1汉字 = 16bit（比特位）
    • 1byte = 8bit（比特位）
    • 1KB = 1024bytes（字节）
    • 1024KB = 1MB
    • 1024MB = 1GB
    • 1024GB = 1TB
    • 1024TB = 1PB
    • 1024PB = 1EB
  • 实际生产厂商容量换算：
    • 1KB = 1000bytes（字节）
    • 1000KB = 1MB
    • 1000MB = 1GB
    • 1000GB = 1TB
    • 。。。
