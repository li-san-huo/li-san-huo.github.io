初识CSS

  • CSS:层叠样式表， Cascading style sheet.
  • 作用:控制网页的外观(修饰和美化html)
  • 语法：选择器{属性1：属性值1；属性值2；属性值3；…}

样式表的种类

  • 行CSS内样式表

  • 内嵌样式表-内部样式表

  • 外部样式表

样式表的优先级

  • 行内样式优先级最高
  • 其他的哪个后加载，哪个优先级高

  1. 选择器：也叫选择符，用于选择页面上要添加样式的标签。

    • 基础选择器：单个选择器

      • 全局选择器（通配符选择器）-选择页面上所有的标签

      • 标签选择器（元素选择器）：选择页面上的同种标签

      • ID选择器：选择页面上的唯一的一个元素

      • 类别选择器：选择页面上class命名相同的标签

      • 基础选择器的优先级

    • 复合选择器：由两个或者是多个基础选择器组合而成的选择器

      • 群组选择器（集体声明）：多个选择器设置同样的样式

      • 后代选择器：用空格连接，选择指定标签元素下的后辈元素

      • 子选择器：用>连接，选择指定元素的第一代子元素

  • 字体属性总结

    属性 表示 注意点
    font=size 字体大小 通常单位用px不可省略
    font-family 字体类型 实际工作中按约定来写
    font-weight 字体加粗 加粗设为bold或者700，正常设为normal或者400
    font-style 字体倾斜 倾斜italic，正常normal
    font 字体复合 1、必须按顺序书写2、大小和字体不可忽略
