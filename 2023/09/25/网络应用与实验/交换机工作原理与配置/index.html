交换机工作原理与配置

一、知识回顾

  • 数据链路层的作用

  • 以太网的发展史

    • 2.94M实验性以太网
    • 10M以太网
    • 100M以太网
    • 1000M以太网

  • 以太网MAC地址组成

    • 定义：通过MAC地址识别以太网中的每个设备（PC）

    • 以太网MAC地址是全球唯一

    • 通过48个比特位表示（前24个比特位表示厂商；后24个比特位表示唯一编码）；其中前24个比特位中的第8位，若是0，表示单播地址；若是1，表示组播地址。

    • MAC地址，又称为“物理地址”。MAC地址焊接在网卡芯片上，用于以太网的数据传输，同时标识以太网中的每一台PC。

  • 以太网的帧格式

    • 以太网帧格式部分组成——目的地址（目前主机MAC地址）；源地址（源主机MAC地址）；数据（46~1500字节）；类型（上层对应的协议，一般指IP协议，即四个支撑协议（icmp/igmp/arp/rarp）；帧校验序列。

二、交换机

  1. 交换机产品系列

    • CISCO 2960（入门级交换机）
    • CISCO 3560（企业级交换机）
    • CISCO 4500（企业模块级交换机）
    • CISCO 6500（大型企业、电信级交换机）

  2. 交换机工作原理

    • 初始化（给交换机进行加电自检）
    • 学习MAC地址（A主机的源MAC地址）
    • 广播（广播B主机的MAC地址）
    • 再学习MAC地址（主机B回应的同时，交换机学习主机B的源MAC地址）
    • 单播（实现主机A到主机B的单播通信）

  3. 交换机工作原理总结

    • 交换机始终学习的是各主机的源MAC地址
    • 交换机把学习到的MAC地址记录到MAC地址表中，记录的具体信息：MAC地址＋对应接口
    • 交换机MAC地址表中的信息不是永久存在的，是临时的，老化时间是３００秒（５分钟）。

  4. 交换机以太网接口的工作模式

    • 单工——两个数据站之间只能进行单向数据传输（类似话筒与扬声器）

    • 半双工——两个数据站之间双向数据传输，但不能同时进行（类似于对讲机）

    • 全双工——两个数据站双向且同时进行数据传输（类似于电话或手机）

  5. 交换机以太网的接口速率

    • 接口速率——自动协商、10M、100M、1000M
    • 接口速率设置时，两端接口必须保持一样
    • 接口速率协商失败，导致网络链路无法正常通信

  6. 交换机的配置方法

    • 通过Console配置线链接PC的COM口实现对交换机的配置（传统配置方法）

    • 直接通过交换机的CLI(Command line interface)命令行接口对交换机进行配置

  7. 交换机命令行的工作模式

    • 用户模式——查看交换机相关信息
    • 特权模式——查看或修改相关信息
    • 全局模式——主要用于对交换机全局配置信息的修改
    • 接口模式——主要用于对交换机接口进行相关的配置

  8. 交换机命令行的帮助

    • ？显示当前工作模式下可以使用的命令列表及参数
    • Tab 命令补齐或命令提示
    • ctrl + A 将光标定位到命令行的开始位置
    • ctrl + E 将光标定位到命令行的结束位置

  9. 交换机命令行工作模式的切换方式

    • 交换机的工作模式

      • 用户模式：Switch>
      • 特权模式：Switch#
      • 全局模式：Switch(config)#
      • 接口模式：Switch(config-if)#

    • 交换机工作模式的切换方式

      • 从用户模式进入或切换到特权模式的命令：enable可简写为en

      • 从特权模式进入或切换到全局模式的命令：configure terminal可简写为 conf t

      • 从全局模式进入或切换到接口模式的命令：interface fastEthernet 0/1可简写为 int f0/1，总结：从用户 模式进入到接口模式，必须一级一级的进行切换。

      • 从接口模式退回到全局模式的命令：exit

      • 从全局模式退回到特权模式的命令：exit 或 end 或 快捷方式：ctrl + z

      • 从特权模式退回到用户模式的命令：exit（只适用于模拟器，真实交换机设备使用命令是disable）

      • 从接口模式直接退回到特权模式的命令：end

  10. CISCO模拟器的使用

    • 绘制网络拓扑图与应用各种网络设备
      • 选择对应的网络设备（交换机、路由器、防火墙、服务器、PC）
      • 选择链接对应网络设备的线缆（配置线、直通线、交叉线等）
      • 对网络设备进行逐一配置（配置IP地址、接口地址等）
      • 测试并验证最终的结果
    • 模拟器网络设备的分类
      1. 交换机
        • 二层交换机：cisco 2960（入门级交换机）
        • 三层交换机：cisco 3560（企业级交换机）
      2. 路由器
        • 路由器型号：cisco 2811、2911等
      3. 防火墙
        • 防火墙系列：cisco 5055
      4. 终端设备
        • 个人PC：台式机、笔记本
        • 服务器：Server-PT
      5. 线缆分类
        • 配置线
        • 直通线
        • 交叉线
    • 模拟器的辅助功能
      • 网络设备添加标识——place note
      • 删除网络设备图标
      • 放大与缩小网络设备图标
      • 选择网络设备或绘制拓扑图，进行移动操作

  11. 以太网中网络设备常用命令

    • ping 测试网络的连通性
    • ipconfig 查看windows系统中，对应网卡配置的ip地址信息
    • ifconfig 查看linux系统中，对应网卡配置的ip地址信息
    • show mac-address-table dynamic 查看交换机中MAC地址表的信息（特权模式）

  12. 网络设备查询注意事项

    • 在特权模式使用show命令，例如：Switch# show mac-address-table dynamic
    • 在全局模式使用show命令，例如：Switch(config)# do show mac-address-table
    • 在接口模式使用show命令，例如：Switch(config-if)#do show mac-address-table

  13. 交换机双工模式配置与速率配置

    • 以太网接口双工模式配置
      • 交换机双工模式在配置时，若交换机两端对应接口的双工模式不一样时，会造成严重的丢包现象，导致 网络无法正常通信。
      • 实际配置交换机接口双工模式采用全双工，交换机两端接口双工模式必须配置一样，使用的命令： duplex full
    • 以太网接口速率配置
      • 交换机速率配置时，若交换机两端对应接口的速率不一样时，直接导致网络无法正常通信。
      • 实际配置交换机接口速率采用100/1000速率，交换机两端接口速率必须配置一样，使用命令：speed 100/1000
    • 查看交换机接口详细信息使用的查询命令
      • show interface f0/24 或者 sh int f0/24（在特权模式执行命令）

  14. 交换网络播的类型

    • 单播：一对一，传输效率最高，延迟最小
    • 组播：一对多，传输效率其次，延迟较小
    • 广播：一对多，传输效率最低，延迟最大
