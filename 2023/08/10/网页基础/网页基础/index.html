网页

是构成网站的基本元素，它通常由图片、链接、文字、声音、视频等元素组成。index.html

HTML

  • 超文本标记语言（Hyper Text Matkup Language），用来描述网页的一种语言。

  • 不是一种编辑语言，而是一种标记语言（markup language）

网页的形成

  • 前端人员开发代码—–> 浏览器显示代码（解析、渲染）—–> 生成最后的Web页面

常用浏览器

  • 作用：是网页显示、运行的平台。
  • 五大浏览器：IE、火狐(Firefox)、谷歌(Google)、苹果(Safari)和欧朋(Opera)。
  • 浏览器内核：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。
  • chrome的内核Blink

web标准

是由W3C组织和其他标准化组织制定的一系列标准的集合。

  1. 结构:html
  2. 样式:css
  3. 行为:js

html语法

基本结构

标签的属性：

  1. 属性：用来描述标签的特征
  2. 属性值：为属性赋的值
  3. 特点：
    • 标签可以拥有多个属性，必须写在标签名的后面。
    • 属性之间不分先后顺序，标签名与属性、属性与属性之间均以空格分开。
    • 属性采取键值对的格式，即 key=“value”的格式，属性=“属性值”

图像标签<img>行内元素(特殊)

  • 定义HTML页面中的图像

  • 属性

    属性 属性值 说明
    src 图片路径 必须属性
    alt 文本 替换文本，图像不能显示时的文本
    title 文本 提示文本，鼠标悬停时显示的文字
    width 像素 设置图像的宽度
    height 像素 设置图像的高度
    border 像素 设置图像的边框粗细

路径

  • 路径：文件或者文件夹存放的位置

  • 路径分为：相对路径和绝对路径

  • 绝对路径：完整的描述文件位置的路径就是绝对路径

    • 从磁盘开始写，如：C:\Users\Administrator\Desktop\web1\1.png
    • 完整的网络地址，例如“http://www.baidu.com“

  • 相对路径：以引用文件所在位置为参考基础，而建立出的目录路径。

    相对路径 符号 说明
    同一级路径 图像文件位于HTML文件同一级 如<img src="pic.png>"
    下一级路径 / 图像文件位于HTML文件下一级 如<img src="images/pic.png>"
    上一级路径 ../ 图像文件位于HTML文件上一级 如<img src"../pic.png">

超链接标签

  • <a>标签用于定义超链接，作用是从一个页面链接到另一个页面。行内元素

  • 语法格式：<a href="跳转目标" target="目标窗口的弹出方式"> 文本或图像 </a>

  • 属性

    属性 作用
    href 用于指定连接目标的url地址
    target 指定页面的打开方式，_self 在原窗口打开(默认值)，_blank 在新窗口中打开
    titile 提示属性

  • 超链接的分类

    • 内部链接:网站内部页面之间的相互链接.直接链接内部页面名称即可，例如< a href="index.html"> 首页</a>。

    • 外部链接：例如< a href="http:// www.baidu.com "> 百度</a>。

    • 空连接：如果当时没有确定链接目标时，< a href="#"> 首页 </a>。

    • 下载链接：如果href里面地址是一个文件或者压缩包，会下载这个文件。

    • 锚点链接：

      • 锚点命名id="p1"

        链接到锚点<a href="#p1"></a>
