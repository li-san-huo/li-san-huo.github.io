网页

是构成网站的基本元素，它通常由图片、链接、文字、声音、视频等元素组成。index.html

HTML

  • 超文本标记语言（Hyper Text Matkup Language），用来描述网页的一种语言。

  • 不是一种编辑语言，而是一种标记语言（markup language）

网页的形成

  • 前端人员开发代码—–> 浏览器显示代码（解析、渲染）—–> 生成最后的Web页面

常用浏览器

  • 作用：是网页显示、运行的平台。
  • 五大浏览器：IE、火狐(Firefox)、谷歌(Google)、苹果(Safari)和欧朋(Opera)。
  • 浏览器内核：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。
  • chrome的内核Blink

web标准

是由W3C组织和其他标准化组织制定的一系列标准的集合。

  1. 结构:html
  2. 样式:css
  3. 行为:js

html基本标签1

标题字标签

<h1>-<h6> 块元素

  • 作为标题使用，并且依据重要性递减
  • 属性：align:left/right/center
  • 特点
    • 文字加粗
    • 字号随着数字变大依次变小
    • 独占一行

段落标签

<p></p>块元素

  • 可以把HTML文档分割为若干段落。
  • 属性： align:left/right/center/justify
  • 特点：
    • 文本在一个段落中会根据浏览器窗口的大小自动换行
    • 段落和段落之间有空行
    • 独占一行

换行标签

<br>

  • 强制换行
  • 特点：
    • 单表签、空标签

水平线标签

<hr>

  • 属性：size（粗细）

    ​ width（宽度） 像素 百分比

    ​ color（颜色）

    ​ align（对齐方式）：left center（居中，默认值）right

文本格式化标签——行内元素

  • 突出重要性，比普通文字更重要。

div和span标签

  • <div> 和 <span> 是没有语义的，它们就是一个盒子，用来装内容的。
  • 特点：
    • <div>标签用来布局，但是现在一行只能放一个-块元素
    • <span>标签用来布局，一行上可以多个-行内元素

图像标签<img>行内元素(特殊)

  • 定义HTML页面中的图像

  • 属性

    属性 属性值 说明
    src 图片路径 必须属性
    alt 文本 替换文本，图像不能显示时的文本
    title 文本 提示文本，鼠标悬停时显示的文字
    width 像素 设置图像的宽度
    height 像素 设置图像的高度
    border 像素 设置图像的边框粗细

路径

  • 路径：文件或者文件夹存放的位置

  • 路径分为：相对路径和绝对路径

  • 绝对路径：完整的描述文件位置的路径就是绝对路径

    • 从磁盘开始写，如：C:\Users\Administrator\Desktop\web1\1.png
    • 完整的网络地址，例如“http://www.baidu.com“

  • 相对路径：以引用文件所在位置为参考基础，而建立出的目录路径。

    相对路径 符号 说明
    同一级路径 图像文件位于HTML文件同一级 如<img src="pic.png>"
    下一级路径 / 图像文件位于HTML文件下一级 如<img src="images/pic.png>"
    上一级路径 ../ 图像文件位于HTML文件上一级 如<img src"../pic.png">
